## General info

A simple app to check Weather

### Project assumptions
1. We have possibility to check weather in any location
2. We use Api from Open Weather

## Technologies
* TypeScript
* Angular
* SCSS
* REST
* Firebase

## Setup
To run this project, clone repo and install it locally using npm:

```
$ npm install
```

Next in terminal use:
```
$ npm start
```
